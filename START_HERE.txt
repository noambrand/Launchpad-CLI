INSTALLATION:

  ⚠ The installer may close open terminal windows during setup.
    Save your work in any open terminals before running the installer.

  Windows:
    1. Double-click ClaudeCode_Launchpad_CLI_Setup.exe (Run as Administrator)
    2. Follow the wizard - choose your language, click Next
    3. The installer auto-detects what you have and skips it

  macOS:
    1. Double-click ClaudeCode_Launchpad_CLI_Setup.pkg
    2. Follow the prompts - Homebrew, Node.js, Git, and Claude Code
       are installed automatically

DAILY USE:

  ★ RECOMMENDED: Pin it to your taskbar (one time, ~10 seconds)
    So you never have to hunt for it on the desktop again:
      1. Click Start and type:  ClaudeCode Launchpad CLI
      2. Right-click the result → "Pin to taskbar"
         (Windows 11: right-click → More → Pin to taskbar, or
          choose "Open file location", then drag the icon to the taskbar)
      3. Done — click the blue Claude icon on your taskbar to launch.

  Desktop shortcut:
    "ClaudeCode Launchpad CLI"  → pick a project folder and start Claude Code

  Or: Start Menu → ClaudeCode Launchpad CLI (Windows)
  Or: Right-click any folder → "Open with ClaudeCode Launchpad CLI" (Windows)
  Or: Open Terminal and run 'claude' (macOS)
